This is part of a larger project. It lets a user look up callsign prefixes
and suffixes. It's built with Flutter and deployed on GitHub Pages automatically
with GitHub actions.

The website is available at [cq.jkob.cc/#/callsign](https://cq.jkob.cc/#/callsign).

## Source code

The source code is available on [GitHub](https://github.com/jakobkordez/ham-tools).
