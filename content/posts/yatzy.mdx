Firstly written in Flutter, later rewritten in Next.js and finally rewritten in SvelteKit, this project is a simple Yatzy scorecard.
It's a simple web app that allows you to use a phone or tablet as a scorecard for the game of Yatzy.

The website is available at [yatzy.jkob.cc](https://yatzy.jkob.cc/).

The source code is available on [GitHub](https://github.com/jakobkordez/yatzy).
